Markdown Preview & Editor

Turn any local Markdown file into a focused browser workspace. Review the rendered document, edit text in place or switch to source, and save without sending content anywhere.

Open in Codex View source

What it does

Markdown review without context switching.

Codex launches a purpose-built localhost workspace around the file you are already working on.

Faithful preview

Render headings, images, links, code, blockquotes, and GitHub-flavored tables in a clean reading layout.

Two ways to edit

Edit rendered text directly with autosave, or switch to the full Markdown source when precision matters.

Local and guarded

Loopback-only access, one-file write scope, version conflict checks, atomic saves, and a strict content policy.

How it works

Ask. Review. Refine.

Point Codex to a file

Ask Codex to use $preview-edit-markdown with the local report or README you want to inspect.

Review in the browser

The plugin finds a free local port, renders the document, and checks its relative images, links, and tables.

Edit with confidence

Make visual or source changes while conflict detection and atomic writes protect the file on disk.

Your document stays on your machine

The bundled service binds only to 127.0.0.1, requires no account, and never uploads document content.
