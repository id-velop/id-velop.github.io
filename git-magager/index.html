Git Magager

Clone repositories from GitHub and GitLab with one click. Choose a destination, let the local companion do the terminal work, and keep your momentum.

Add to Chrome View source

What it does

From repository to local folder, faster.

Git Magager removes the repetitive steps between finding a repository and starting work on it.

One-click clone

Adds a focused clone action on supported GitHub and GitLab repository pages.

Choose your folder

Use the native folder picker and send the repository exactly where it belongs.

Local by design

Your repository URLs and paths are handled on your device, not shipped to a remote service.

How it works

Three steps. Then build.

Open a repository

Visit any supported GitHub or GitLab repository page.

Choose your clone option

Pick HTTPS or SSH, then select the local destination folder.

Keep moving

Git Magager hands the command to your local companion and starts the clone.

Your data stays local

Git Magager does not collect, store, or transmit personal information to external servers.
