# Reference

This directory stores reusable Markdown reference documents for `md-illustration-generator`.

Add confirmed component guide examples here when they should become reusable baselines for future illustration generation. The main skill reads this directory before falling back to a target repository's `Component/` directory.

Guidelines:

- Keep files as source Markdown references, not generated previews or screenshots.
- Prefer final or reviewed examples over in-progress drafts.
- Preserve the original filename when possible, for example `Badge.md` or `Segmented_Component_Guide.md`, so references in `SKILL.md` resolve directly.
