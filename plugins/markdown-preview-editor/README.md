# Markdown Preview & Editor

Edit local Markdown directly in the browser. Every change saves automatically.

## Use it

Ask Codex:

> Use `$preview-edit-markdown` to open `path/to/file.md`.

Click the text and type. There is no mode to switch and no save button to manage.

## Local by default

The editor runs on `127.0.0.1` and only updates the Markdown file you opened.

## Requirements

- Codex
- Node.js 18+
